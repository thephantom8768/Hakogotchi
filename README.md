# Hakotchi
Hakotchi is a tamagotchi simulates a living creature that depends on its environment to survive. Players must carefully manage food, entertainment, light levels, temperature, and humidity to keep their pet healthy and happy. It turns virtual pet care into a challenging environmental management experience, basically tamagotchi on hardcore mode.
